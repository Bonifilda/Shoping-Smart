 ShopSmart : Product Discovery & Wishlist
ShopSmart is a simple, responsive web application that allows users to discover products, 
filter by categories, and save their favorites in a wishlist. Built with HTML, CSS (Tailwind), and JavaScript, it fetches real product data from the Fake Store API.

 Features
- Product Discovery: Browse products with search and category filters.
- Wishlist: Save your favorite items locally using `localStorage`.
- Responsive Design: Works seamlessly on mobile, tablet, and desktop.
- Dynamic Loading: Shows loading spinner, error messages, and no-results notifications.
- Navigation: Easy navigation between Home, Wishlist, and About pages.
- Real-time API Integration: Fetches products and categories dynamically from the API.

 Technologies Used
HTML5 & CSS3
Tailwind CSS
JavaScript (ES6 Modules)
Font Awesome for icons
Fake Store API for product data
